# Udemy-clone
This is a basic Udemy homepage clone created using only HTML and CSS. It replicates the front-end structure of the original Udemy website to help practice layout design, responsiveness, and clean code structuring.

# Features
Static UI inspired by Udemy’s homepage

Responsive design using Flexbox 

Navigation bar with logo, links, and search box

Course cards with images, titles, and pricing

Footer section with links and social media placeholders

# Technologies Used
HTML5

CSS3
